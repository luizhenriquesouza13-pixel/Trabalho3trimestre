1) Classificador de número (if/else)

Objetivo: Ler um número inteiro e informar se é positivo, negativo ou zero, além de verificar se é par ou ímpar.

Conceitos: Estruturas condicionais (if/else), operadores aritméticos, módulo %.

2) Soma e média até sentinela (while)

Objetivo: Ler números sucessivos até o usuário digitar -1, e calcular quantidade, soma e média.

Conceitos: Laço de repetição while, contadores, acumuladores, sentinela.

3) Tabuada (for)

Objetivo: Ler um número inteiro e imprimir sua tabuada de 1 a 10.

Conceitos: Laço for, multiplicação, formatação de saída.

4) Vetor — Maior, menor e média

Objetivo: Ler 10 números em um vetor e mostrar o maior, menor e a média.

Conceitos: Vetores, laços for, comparação de valores, cálculo de média.

5) Contar vogais em uma string (for + strings)

Objetivo: Ler uma linha de texto e contar quantas vogais existem, ignorando maiúsculas/minúsculas.

Conceitos: Strings, loops, funções de manipulação de caracteres (tolower), operadores lógicos.

6) Teste de primo (if/else + for)

Objetivo: Verificar se um número ≥ 2 é primo; se não, mostrar seu menor divisor.

Conceitos: Laço for, condicionais if/else, lógica de divisibilidade.

7) Menu RH: INSS ou hora extra

Objetivo: Criar um menu para calcular INSS (8% do salário) ou valor das horas extras (50% a mais), repetindo até o usuário escolher sair.

Conceitos: Estruturas de repetição (do/while), menu com switch, operações financeiras simples.

8) Vetor de salários — Listar ou média

Objetivo: Ler N salários e permitir via menu listar todos ou mostrar a média, sem alterar os valores.

Conceitos: Vetores, laços, menu com switch, cálculo de média.

9) Vendas (matriz 2×2) — Totais

Objetivo: Ler uma matriz 2×2 (2 filiais × 2 meses) e, via menu, mostrar total por filial, por mês ou total geral.

Conceitos: Matrizes, somas de linhas e colunas, menus, laços ou cálculo direto.

10) RH — Cadastro mínimo (id + salário)

Objetivo: Cadastrar N funcionários com id e salário em vetores paralelos; menu para listar todos ou mostrar maior salário com id.

Conceitos: Vetores paralelos, laços, busca de máximo, menu interativo.
